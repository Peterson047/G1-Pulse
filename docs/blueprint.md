# **App Name**: G1 Pulse

## Core Features:

- RSS Feed Parsing: Parse the G1 Brasil RSS feed (https://g1.globo.com/rss/g1/brasil/) to extract news articles.
- Article Preview: Display each news article with a preview image, title, and short description.
- Category Filtering: Allow users to filter news articles by category.
- Full Article View: Provide a dedicated page for reading the full content of a selected news article.
- AI Article Summary: Generate a concise summary of each news article using generative AI, for quick reading.

## Style Guidelines:

- Primary color: Dark blue (#223142) to convey trust and reliability.
- Background color: Very light blue (#F0F4F8).
- Accent color: Bright orange (#E07A5F) for important callouts and links.
- Body font: 'Inter' sans-serif, for clean and modern readability.
- Headline font: 'Space Grotesk' sans-serif, for a contemporary look.
- Use flat, minimalist icons for categories and navigation.
- Implement a grid-based layout for presenting news articles in a structured manner.
- Use subtle transitions and loading animations for a smooth user experience.
